# Basik
## Description

The intention of this project is to create a basic API that will serve the data from static md files. (the backend of a blog)

This runs using [Nest](https://github.com/nestjs/nest) framework

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
